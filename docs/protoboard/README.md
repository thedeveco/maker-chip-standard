# Protoboard Project

[Protoboard](https://protoboard.xyz) is where the MakerChip+ PCB is designed and validated, and it is the **single best starting point for a `3.x` build**. Instead of chasing parts and specs across a dozen tabs, the board hands you the entire system in one place.

## Open or import it

- **Live board:** https://alpha.protoboard.xyz/board/pCOh7ogceIT44E7xwzadPOtLD8I3/_w6jix_-YGsyXYv-8ICmD
- **Importable file:** [`maker-chip-pcb-v4.protoboard`](maker-chip-pcb-v4.protoboard) - download it and open at [alpha.protoboard.xyz](https://alpha.protoboard.xyz) to load the whole project locally.

## What you get for free

- Every component with its **datasheet** attached
- A live **BOM** with vendors, pack sizes, and per-board cost
- The **system / harness diagram** - how the 8-pad interface wires to the ATtiny85 and the LEDs
- **DRC** validation and design notes

This is why we point builders at Protoboard first: fork the board, swap a part, and the BOM, datasheets, and connection checks update with you.
