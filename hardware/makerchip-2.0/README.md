# MakerChip+ 2.0 - Chip and Dock CAD

Mechanical CAD for the `2.x` magnetic tier of the [MakerChip+ standard](../../spec/README.md): the chip and its mating dock.

This is the [magnetic standard](../../spec/magnetic.md) realized as solid geometry. The polarized magnet pair lets the chip seat in exactly one orientation against the dock.

## Files

| File | What it is |
|---|---|
| `chip.step` | The MakerChip+ 2.0 chip (40 mm, dual 15 x 2 mm magnet pockets) |
| `dock.step` | The mating dock (inverse magnet pattern for snap-in alignment) |
| `chip.3mf` | Print-ready MakerChip+ 2.0 chip |
| `chip-clear.3mf` | Print-ready chip, clear-window variant |
| `hat-clip-dock.3mf` | Print-ready hat-clip dock (>= 2.0) |

## Source

Both parts are modeled in the public live Onshape project:
https://cad.onshape.com/documents/66dffec1c63767f9fad75111/w/110fc95ea174063827d47415/e/8618be80ab8abbf698550313
